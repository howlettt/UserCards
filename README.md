# User Cards
React, Material UI, & Vite demo site

## Setup
- Run `npm install`

## View Site
- Run `npm run dev`
- Open `http://localhost:5173/`

## Unit Tests
- Run `npm run test`

## Safetests
- Install Playwright by running `npx playwright install`
- Start the site by running `npm run dev`
- Run `npm run safetest` in another shell window
